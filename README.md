# Django-Ecommerce-Store
 Admin Dashboard (Django)

A professional Django-based Admin Dashboard for managing brands and products.
Built with clean UI, responsive layout, and secure admin access.

 Features

* Add, edit, and delete brands easily
* Upload and preview brand logos
* Responsive sidebar navigation
* Simple and modern UI with cards and tables
* Secure login/logout system

 Tech Stack

* Django (Python)
* HTML5, CSS3
   Django Template Engine

 Setup
1. Clone repo and install dependencies
2. Run `python manage.py migrate`
3. Create superuser: `python manage.py createsuperuser`
4. Start server: `python manage.py runserver`
5. Access dashboard at `http://127.0.0.1:8000/`
